int  fd

RunloopStreamRef is a special type of watcher designed to watch stream style file descriptors (such a sockets and pipes) to detect when such an fd is ready to perform a read operation or ready to perform a write operation without BLOCKING. This type of watcher is intended for situations where a single thread may be performing both read and write operations on the same fd and on multiple file descriptors.
