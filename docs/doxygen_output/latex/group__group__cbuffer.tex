\doxysubsection{Detailed Description}
A Cbuffer is a contiguous memory allocation that can expand as required. It maintains\+:
\begin{DoxyItemize}
\item the length of the total memory allocation,
\item the length of the used part of the memory allocation
\item always terminates the used portion with a \textquotesingle{}\textbackslash{}0\textquotesingle{} which is not counted as part of the used portion so that the used portion is always a valid cstr 
\end{DoxyItemize}
